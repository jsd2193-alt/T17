# Project Skill-Sync : AI 자율 교육 생태계

국방 AI 활용 아이디어 경연대회 제안을 위한 전문 프레젠테이션 프로젝트입니다.

## 🚀 개요
본 프로젝트는 대한민국 군의 '인력 소요 위기'를 해결하기 위해, 신병 및 특기병 교육의 주체를 인간에서 인공지능으로 전환하는 혁신적인 방안을 제안합니다.

## 🛠 주요 기술
- **Holo-Sergeant**: 생성형 AI 기반 XR 가상 교관
- **Optic-Audit**: 컴퓨터 비전 활용 지능형 자세 교정
- **Adaptive-Drill**: AI 기반 개인별 적응형 커리큘럼

## 📂 프로젝트 구조
- `index.html`: 메인 프레젠테이션 페이지
- `style.css`: 고급 기술 디자인 (Military-Futuristic)
- `main.js`: 인터랙티브 효과 및 애니메이션 로직
- `OFFICIAL_PROPOSAL.md`: 상세 기획서 (서술형)

## 🌐 실행 방법
`index.html` 파일을 브라우저에서 열어 확인하시기 바랍니다.
